\section{CONCLUSION}
\label{sec:conclusion}

The development and evaluation of \textbf{SafeSurf} demonstrate that integrating asynchronous infrastructure analysis with a multi-model ensemble approach significantly improves the robustness, accuracy, and speed of real-time phishing detection. By achieving a classification accuracy of \textbf{97.35\%} on the PhiUSIIL dataset \cite{ref10}, the framework effectively addresses the limitations of traditional static blacklists and single-model classifiers. The layered architecture—specifically the DNS Guard and the concurrent feature extraction pipeline—ensures that the system provides high-performance protection without compromising the user experience within a browser environment.

This research underscores the importance of looking beyond the URL string and evaluating the underlying internet infrastructure. Features such as domain registration age, SSL certificate validity, and name server distribution provide critical context that is difficult for attackers to spoof, thereby enhancing the system's resistance to zero-day threats and sophisticated obfuscation techniques.
