Phishing remains a significant threat to global cybersecurity, necessitating advanced detection systems that can adapt to rapidly evolving attack methods. This paper proposes \textbf{SafeSurf}, a real-time phishing detection framework that leverages a multi-model ensemble approach and asynchronous infrastructure analysis. By extracting multi-dimensional features from DNS records, SSL certificates, and RDAP registration data, SafeSurf provides a comprehensive evaluation of website integrity. The system employs a deterministic DNS Guard module followed by a sophisticated VotingClassifier ensemble—combining Logistic Regression, LinearSVC, Random Forest, HistGradientBoosting, and Calibrated XGBoost—to achieve a robust classification accuracy of 97.35\%. Furthermore, the framework includes a browser extension that offers instantaneous safety verdicts, ensuring minimal user latency. Our results demonstrate that SafeSurf significantly outperforms standalone machine learning models in precision and recall, providing an effective and scalable solution for proactive web protection.
